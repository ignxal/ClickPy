# ClickPy
Requirements
* Phyton3.7+
* Pip

# Instalation
Go to project folder
```
$ cd ClickPy
```
Install
```
$ pip install pandas openpyxl python-dotenv
```
# Configuration
* Copy your token inside .env file
* Copy your URL inside get_TodayUrl() WITHOUT the endtime nums

# Run app
<code>$ phyton app.py</code>
